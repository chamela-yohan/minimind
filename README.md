# MiniMind

A neural network engine built from scratch in C++ — no ML libraries, no shortcuts.

## What it does
Implements the full forward pass, backpropagation, and gradient descent
from first principles. Trains on the XOR problem as a demo.

## Why
Understanding AI from the inside out. C++ is the language PyTorch is written in.

## Status
🔧 In progress — building step by step.

## Build
```bash
g++ main.cpp -o minimind
./minimind
```
